키 무료 발급

에디터 완벽 사용 가이드

One-Page Editor의 핵심 기능과 100% 활용 방법을 상세히 안내해 드립니다.

1. AI 활성화를 위한 API 키 설정

AI 자동 생성 기능을 사용하려면 제미나이(Gemini) API 키가 필요합니다. 우측 상단의 톱니바퀴 아이콘을 클릭하여 무료로 발급받은 키를 입력해 주세요.

2. 아이디어 한 줄로 기획서 자동 생성

상단 검색창에 게임의 핵심 아이디어를 자유롭게 입력하고 [기획서 굽기] 버튼을 클릭하세요. AI가 장르, 타겟층, 코어 루프, 주요 특징(USP)까지 구조화된 기획서를 즉시 작성해 줍니다.

3. 디자인 및 테마 커스터마이징

좌측 설정 패널의 '포인트 컬러 동기화'를 통해 메인 색상을 지정하면 전체 UI의 톤앤매너가 자동으로 어우러지게 변경됩니다. 클래식, 시네마틱, 그리드 3가지 레이아웃 템플릿도 마음대로 교체해 보세요!

4. 디테일 수정 및 컨셉 아트 추가

AI가 작성한 내용 중 수정이 필요한 부분은 좌측 패널의 텍스트 박스에서 직접 편집할 수 있습니다. [이미지 선택] 버튼을 눌러 로컬에 있는 컨셉 아트나 로고 이미지를 업로드해 시각적인 완성도를 높여보세요.

5. 상세 기획서 (GDD) 확장 작성

원페이지 요약본으로 부족하다면 좌측 중앙의 [GDD 자동 작성하기] 버튼을 클릭하세요. 세계관, 시스템, BM을 포함한 심도 있는 문서를 이어서 작성해 줍니다.

6. 고해상도 추출 및 문서 저장

우측 상단의 다운로드 버튼을 통해 완성된 기획서를 고화질 PDF 또는 PNG로 안전하게 추출할 수 있습니다. [SVG] 버튼을 누르면 코드가 복사되니 피그마(Figma)에 바로 붙여넣기하여 작업해 보세요!
